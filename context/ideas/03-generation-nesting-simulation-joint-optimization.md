# Idea 03: 生成-排料-仿真联合优化

## 概述

提出一个**三目标联合优化框架**，同时优化：(1) 版型形状的美观性/功能性，(2) 排料效率（面料利用率），(3) 物理仿真验证的穿着效果。当前这三个环节完全独立——AI生成版型时不考虑排料效率，排料软件不修改版型形状，仿真结果不反馈到版型设计。联合优化可以在**不显著影响穿着效果**的前提下，通过微调版型轮廓曲线来大幅提升面料利用率。

## 指向的Gap

- **Gap 20: 排料优化（Marker Making）** — AI版型生成与排料优化完全脱节，版型形状和排料效率应该被联合优化
- **Gap 26: 生成-仿真闭环的缺失** — 版型生成和物理仿真是完全脱节的两个过程
- **Gap 19: 面料约束的缺失** — AI生成版型时完全不考虑面料幅宽等物理约束

## 推导过程

### 从Gap到Idea的逻辑链

1. **观察**：面料成本占服装制造成本的60-70%，行业平均面料利用率仅75-85%（15-25%浪费）。fashioninsta.ai报告指出"46% fabric reduction possible"。
2. **洞察**：版型的外轮廓曲线有一定的"设计自由度"——在同一穿着效果的约束下，轮廓曲线的微小变化对穿着效果影响很小，但对排料效率影响很大。例如，将一条凸曲线微调为更平直的线段，可能使相邻版片更紧密排列，但穿着差异几乎不可察觉。
3. **方法论**：使用可微优化将三个目标统一——版型形状通过可微参数曲线表示，排料效率通过可微近似（软占用网格）度量，穿着效果通过可微布料仿真评估。

### 关键创新

当前没有任何方法同时考虑这三个目标。这是因为：
- **版型生成**（ML社区）和**排料优化**（运筹学社区）和**布料仿真**（图形学社区）分属不同的学术社区
- 联合优化需要所有三个组件都是可微的，这在技术上最近才变得可能（DiffAvatar等可微仿真方法的出现）

## 详细设计

### 1. 版型参数化表示

使用B-spline参数曲线表示版型轮廓：
- 每个panel的边界由一组B-spline控制点定义
- 控制点坐标是可优化的连续变量
- 约束：panel必须闭合（首尾控制点相连），相缝边长度匹配

### 2. 排料效率的可微近似

精确的2D nesting是NP-hard的组合优化问题，无法直接求梯度。提出两种可微近似：

**方案A：软占用网格（Soft Occupancy Grid）**
- 将面料区域离散为高分辨率网格
- 每个panel通过可微光栅化（differentiable rasterization）投影到网格上
- 排料效率 = 总被占用格子数 / 面料总面积
- 碰撞惩罚 = 重叠格子的重叠度之和
- 所有操作可微——可通过反向传播优化panel位置和形状

**方案B：学习排料策略（Learned Nesting Policy）**
- 训练一个神经网络预测给定版型集的最优排列
- 输入：所有panel的形状描述符
- 输出：每个panel的位置和旋转角度
- 优势：推理时更快，可端到端训练

### 3. 物理仿真验证

使用可微布料仿真（遵循DiffAvatar的方法论）：
- 将优化后的版型缝合并仿真到SMPL身体上
- 计算穿着质量指标：
  - Ease分布（各部位松量）
  - 悬垂角度（drape coefficient）
  - 应力分布（fabric stress）
- 穿着质量指标与原始版型（优化前）的偏差作为约束

### 4. 联合优化目标

$$\min_{\theta} \quad \alpha \cdot L_{nesting}(\theta) + \beta \cdot L_{simulation}(\theta) + \gamma \cdot L_{shape}(\theta)$$

- $\theta$：所有panel的B-spline控制点坐标 + 排料位置/旋转
- $L_{nesting}$：1 - 面料利用率 + 重叠惩罚
- $L_{simulation}$：穿着效果与原始版型的偏差（ease差异 + 悬垂差异）
- $L_{shape}$：版型形状变化量（正则化，防止过度变形）
- $\alpha, \beta, \gamma$：权重超参数，控制各目标的相对重要性

### 5. 优化策略

- 阶段1：固定排料，优化版型形状以改善仿真效果
- 阶段2：固定版型形状，优化排料位置
- 阶段3：联合优化所有变量
- 使用交替优化（alternating optimization）避免陷入差的局部最优

## 参考文献分析

### 核心参考

1. **DiffAvatar** (Li et al., CVPR 2024, arXiv:2311.12194)
   - 证明了可微布料仿真可以用于版型优化
   - "Optimization takes about one minute per iteration" — 仿真速度是瓶颈
   - 本提案可借鉴其仿真框架，但目标函数不同（DiffAvatar优化形状匹配，本提案优化多目标）

2. **Inverse Garment** (Yu et al., CGF 2024, arXiv:2403.06841)
   - 另一个可微仿真优化版型的工作
   - 证明了通过梯度下降优化版型参数的可行性

3. **fashioninsta.ai 行业报告**
   - "46% fabric reduction possible" — 说明AI辅助排料有巨大节省空间
   - 面料成本占制造成本60-70% — 经济激励明确

4. **NP-hard nesting问题** (Wascher et al., 2007)
   - 不规则形状2D nesting的计算复杂度分析
   - 启发式算法（模拟退火、遗传算法）是工业标准
   - 本提案用可微近似绕过NP-hard性

5. **Gerber AccuMark Nest / Lectra Diamino**
   - 工业排料软件，使用启发式算法
   - 不修改版型形状——这是本提案的核心创新点

## 风险与挑战

1. **计算成本极高**：每次迭代需要运行可微仿真（分钟级）+ 排料优化（秒级），联合优化可能需要数百次迭代
2. **可微nesting近似误差**：软占用网格的分辨率限制了精度，可能错过更优的排列方案
3. **多目标权衡的主观性**：$\alpha, \beta, \gamma$的设定取决于用户偏好（更省料还是更好穿），需要交互式调整
4. **版型形状变化的可接受范围**：微调版型轮廓可能改变设计师的美学意图，需要严格的变形约束
5. **面料幅宽约束**：不同面料有不同幅宽（110/140/150cm），排料优化需要将此作为硬约束
