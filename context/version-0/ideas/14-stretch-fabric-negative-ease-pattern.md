# Idea 14: 弹性面料负Ease版型生成

## 概述

开发专门针对弹性面料（针织面料、含氨纶面料等）的版型生成方法，核心创新在于建模**负ease**（negative ease）——即版型尺寸小于身体尺寸10-40%，依靠面料拉伸贴合身体。当前所有版型生成方法都假设面料不可拉伸（isometric），完全无法处理运动服、泳装、内衣、压缩服等弹性服装的版型设计。

## 指向的Gap

- **Gap 13: 非刚性面料的展平失真** — 3D→2D展平算法假设面料不可拉伸，弹性面料的负ease未被建模
- **Gap 19: 面料约束的缺失** — AI不考虑面料弹性属性对版型尺寸的影响

## 推导过程

1. **DiffAvatar明确声明**："represent clothing in 2D pattern space, which ensures developable meshes"——即假设面料不可拉伸
2. **Inverse Garment同样假设**："assumes fabric is inextensible"——展平算法不处理拉伸
3. **市场规模**：全球运动服市场超$1800亿，几乎全部使用弹性面料。泳装、内衣、压缩服等也都依赖弹性面料
4. **核心洞察**：弹性面料版型的关键参数是"版型缩放比"——版型相对身体的缩小百分比，这取决于面料弹性系数和目标压缩力

## 详细设计

### 1. 弹性面料力学模型

将面料弹性建模为方向性应变场：

```
面料应变模型：
  σ(ε) = E · ε + η · ε²  (非线性弹性)

其中：
  σ = 应力 (N/m)
  ε = 应变 (无量纲，= (L_stretched - L_original) / L_original)
  E = 弹性模量 (经向/纬向不同)
  η = 非线性系数

关键参数：
  ε_max = 最大可拉伸率（超过后面料损坏或不舒适）
  ε_comfort = 舒适拉伸范围上限
  Recovery(ε) = 卸载后回弹率
```

### 2. 负Ease计算

```
对于身体表面上的每个点 p：
  body_measurement(p) = 身体在该点的围度/长度
  desired_stretch(p) = 目标拉伸率（基于舒适/功能需求）
  pattern_measurement(p) = body_measurement(p) / (1 + desired_stretch(p))

负ease = pattern_measurement - body_measurement < 0
```

**不同应用场景的目标拉伸率**：

| 应用 | 目标拉伸率 | 负ease |
|------|-----------|--------|
| 运动T恤（宽松） | 5-10% | -5% to -10% |
| 紧身运动衣 | 15-25% | -15% to -25% |
| 泳装 | 20-35% | -20% to -35% |
| 压缩裤 | 25-40% | -25% to -40% |
| 医疗压力袜 | 30-50%（精确控制） | -30% to -50% |

### 3. 可微弹性仿真优化

```
目标：找到版型 P 使得穿着后产生目标压力/合体效果

优化循环：
  1. 初始化版型 P₀ = body_surface × (1 - avg_ease)
  2. 使用弹性材料仿真器模拟穿着（含拉伸变形）
  3. 计算穿着后各部位的实际拉伸率和压力分布
  4. 与目标拉伸率/压力比较，计算损失
  5. 通过可微仿真反传梯度到版型参数
  6. 更新版型 → 回到步骤2
```

**损失函数**：
$$L = \lambda_1 L_{stretch} + \lambda_2 L_{pressure} + \lambda_3 L_{coverage}$$

- $L_{stretch}$：各部位实际拉伸率与目标的偏差
- $L_{pressure}$：压力分布与目标的偏差（对压缩服装重要）
- $L_{coverage}$：确保面料完全覆盖身体（无缺口/过度堆叠）

### 4. 数据与训练

**训练数据生成**：
- 对于多种弹性面料参数 × 多种体型 × 多种拉伸目标
- 使用FEM弹性仿真生成ground truth版型-穿着对
- 预估：100面料 × 200体型 × 5拉伸目标 = 100,000样本

**快速推理模型**：
- 训练神经网络直接预测：(body_shape, fabric_params, target_stretch) → pattern
- 无需每次运行优化循环
- 用优化循环生成的结果作为监督信号

## 参考文献分析

1. **DiffAvatar** (Li et al., CVPR 2024): "ensures developable meshes and manufacturable clothing"——假设不可拉伸
2. **Inverse Garment** (Yu et al., CGF 2024): 纯几何展平，无弹性面料考虑
3. **ASTM D4964**: 针织面料弹性和回弹率的标准测试方法
4. **运动服市场数据**: 全球运动服装市场规模超$1800亿
5. **压缩服装文献**: 医疗压力袜的压力梯度设计有严格的工程规范——可作为参考
6. **SewingLDM** (ICCV 2025): Body-shape conditioning但不处理弹性面料

## 风险与挑战

1. **弹性仿真的稳定性**：大变形弹性仿真（30-50%应变）数值不稳定
2. **面料弹性参数的获取**：用户可能不知道面料的精确弹性参数
3. **舒适度的主观性**：目标拉伸率/压力在一定程度上因人而异
4. **方向性弹性**：针织面料在不同方向弹性差异很大（经向 vs 纬向），增加了问题复杂度
5. **面料疲劳**：反复拉伸后面料弹性下降，版型设计是否应考虑长期使用
