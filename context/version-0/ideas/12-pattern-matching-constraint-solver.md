# Idea 12: 图案对花约束求解器

## 概述

开发一个专门处理**图案对花（Pattern Matching/Alignment）**问题的约束求解器。当使用条纹、格纹、印花等有规律图案的面料时，版片在面料上的放置位置必须保证图案在缝合线两侧对齐——这是工业制版中最困难的约束满足问题之一。当前没有任何AI方法涉及这一问题。

## 指向的Gap

- **Gap 19: 面料约束的缺失——图案对花** — 条纹/格纹面料需要跨版片图案连续，这是极难的约束满足问题
- **Gap 20: 排料优化** — 图案对花约束显著影响排料效率

## 推导过程

1. **工业痛点**：使用格纹面料时，面料浪费率可从常规的15-25%飙升至30-50%，因为版片放置位置受到图案重复单元的严格限制
2. **计算复杂度**：图案对花本质上是在2D排列+图案相位约束下的组合优化问题
3. **AI的优势**：人类版型师依靠经验和空间直觉处理对花，AI搜索可以找到人类难以想象的最优排列

## 详细设计

### 1. 图案表示

将面料图案建模为2D周期性函数：

```
Print(x, y) = F(x mod Tx, y mod Ty, θ)

其中：
- Tx, Ty: 水平/垂直重复周期
- θ: 图案旋转角度
- F: 图案函数（图像或参数化描述）
```

| 图案类型 | Tx | Ty | θ | 约束难度 |
|----------|----|----|---|---------|
| 横条纹 | ∞ | Ty | 0° | 低——只需水平对齐 |
| 竖条纹 | Tx | ∞ | 0° | 低——只需垂直对齐 |
| 格纹（对称） | Tx | Ty | 0° | 中——需要双向对齐 |
| 格纹（非对称） | Tx | Ty | 0° | 高——需要方向匹配 |
| 斜纹 | Tx | Ty | 45° | 极高——旋转约束 |
| 自由印花 | Tx | Ty | any | 极高——完整2D相位匹配 |

### 2. 对花约束形式化

对于每对缝合在一起的边 (eᵢ, eⱼ)：

```
约束：Print(Panel_i(eᵢ(t))) = Print(Panel_j(eⱼ(t))), ∀t ∈ [0, 1]

等价于：
  图案相位在缝合线处匹配
  即：(xᵢ(t) mod Tx, yᵢ(t) mod Ty) = (xⱼ(t) mod Tx, yⱼ(t) mod Ty)
```

### 3. 优化算法

**混合整数规划(MIP)方法**：

决策变量：
- 每个panel在面料上的位置 (xᵢ, yᵢ) ∈ R²
- 每个panel的旋转角度 θᵢ ∈ {0°, 90°, 180°, 270°}
- 图案相位整数偏移 (kᵢ, lᵢ) ∈ Z²

目标：
- 最小化面料使用面积（总面料长度 × 幅宽）

约束：
- 对花约束（缝合线图案匹配）
- 无重叠约束（版片间无碰撞）
- 面料幅宽约束
- Grain line约束（经纬方向正确）

**可微松弛方法**：
- 将离散的相位偏移松弛为连续变量
- 使用soft constraint penalty代替硬约束
- 通过梯度下降优化

### 4. 与版型生成的集成

作为后处理模块：
```
版型生成（任何方法） → 版片集合
    ↓
面料选择（用户选择格纹/条纹面料） → 图案参数
    ↓
图案对花求解器 → 优化排列方案
    ↓
输出：排料图（含对花标记）
```

## 参考文献分析

1. **无AI先行工作**：据调研，没有任何学术论文用AI解决图案对花问题
2. **Gerber/Lectra排料工具**：有基本的条纹/格纹模式，但优化能力有限
3. **2D nesting文献** (Wascher et al., 2007): 标准nesting不考虑图案约束
4. **混合整数规划**：CPLEX、Gurobi等求解器可以处理有约束的排列优化
5. **工业实际**：格纹面料的对花是衡量服装质量的重要标志——高档西装要求肩线、袖笼、口袋等处图案完美对齐

## 风险与挑战

1. **计算复杂度爆炸**：图案约束使得本已NP-hard的nesting问题更加困难
2. **图案信息获取**：用户需要准确描述面料图案的重复单元，这可能需要图像识别辅助
3. **曲线缝合线的对花**：直线缝合线的对花相对简单，曲线缝合线（如袖窿）的对花极其复杂
4. **实用性范围**：仅限于有规律图案的面料，对素色面料无意义
